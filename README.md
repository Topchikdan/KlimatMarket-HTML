# KlimatMarket — HTML Page
**Страница товара для интернет-магазина климатической техники — без фреймворков, только чистый HTML, CSS и JavaScript.**

## ✨ Возможности

- 🖼 Галерея изображений с переключением по миниатюрам
- 🎨 Выбор цвета товара с анимированным индикатором
- 🛒 Кнопка «Добавить в корзину» с ripple-эффектом и toast-уведомлением
- ❤️ Кнопка «Избранное» с toggle-состоянием
- ⭐ Блок рейтинга и отзывов
- 📐 Полностью адаптивная вёрстка (десктоп → планшет → мобильный)
- 🔝 Sticky-шапка с анимированным подчёркиванием при наведении
- 💅 Современный дизайн: карточки, тени, плавные переходы

## 🛠 Технологии

- **HTML5** — семантическая разметка
- **CSS3** — Flexbox, кастомные анимации, медиазапросы, CSS-переменные
- **Vanilla JavaScript (ES6+)** — интерактив без библиотек

## 📁 Структура проекта

```
KlimatMarket-HTML/
├── index.html       # Разметка страницы
├── style.css        # Базовые стили и типографика
├── header.css       # Стили шапки
├── product.css      # Стили карточки товара и анимации
├── media.css        # Адаптивность и брейкпоинты
└── index.js         # Вся интерактивная логика
```

## 🚀 Как запустить

Сборка не нужна — просто открой файл в браузере:

# Клонировать репозиторий
git clone https://github.com/your-username/KlimatMarket-HTML.git

# Перейти в папку
cd KlimatMarket-HTML

# Открыть в браузере
open index.html

Или перетащи `index.html` прямо в браузер — и всё готово.
